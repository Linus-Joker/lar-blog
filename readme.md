# linus-blog

需在本機建構伺服器環境

# Requirements

- PHP 7.2.10
- HTTP server with PHP support (Apache)
- Composer
- A supported database: MySQL

# Getting Started

clone and go into this repository

$ git clone https://github.com/Linus-Joker/linus-blog.git

install dependencies and run

$ npm install

$ composer install
